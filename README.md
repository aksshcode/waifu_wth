# waifu_wth

A waifu image swiper built with [Next.js](https://nextjs.org) 16.

Fetches waifu images from [nekos.best](https://nekos.best) and lets you swipe left/right to browse. Pre-fetches the next image in the background so swipes are instant.

## Features

- Swipe or use arrow keys to browse waifu images
- Pre-fetches next image for instant transitions
- Entrance/exit animations with motion
- Floating navigation bar
- Dark/light mode

## Getting Started

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
bun run build
```
